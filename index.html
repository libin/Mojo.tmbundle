Mojo.tmbundle by libin

TextMate Bundle for Palm Mojo SDK

This is the TextMate Bundle to help you get up to speed on Palm Mojo Development. It's still on a very early stage. Feel free to fork it!

Dependencies

Palm Mojo SDK It's Free and you can get it from: http://developer.palm.com/index.php?option=com_content&view=article&id=1597

Install

To install with Git: mkdir -p ~/Library/Application\ Support/TextMate/Bundles cd ~/Library/Application\ Support/TextMate/Bundles git clone git://github.com/libin/Mojo.tmbundle.git "Mojo.tmbundle" osascript -e 'tell app "TextMate" to reload bundles'
