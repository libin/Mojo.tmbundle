Mojo.tmbundle by Libin Pan

TextMate Bundle for Palm® Mojo™ SDK

This is the TextMate Bundle to help you get up to speed on Palm Mojo Development. It's still on a very early stage. Feel free to fork it!

Features

  • Commands
    1. Launch the Palm Emulator
    2. Package Current Project
    3. Generate new Scene
    4. Install Selected Package on Emulator
    5. Uninstall Selected Package from Emulator
    6. Install Selected Package on Device
    7. Uninstall Selected Package from Device
    8. List Emulator Installed Applications
    9. List Device Installed Applications
    10. Launch Application on Emulator
    11. Launch Application on Device
    12. Runs Application on Emulator with Inspector
    13. Runs Application on Device with Inspector
    14. Close Application on Emulator
    15. Close Application on Device
    16. Close and relaunch Application on Emulator
    17. Close and relaunch Application on Device

Dependencies

Palm® Mojo™ SDK

It's Free and you can get it from: Developer.Palm.com

JSON Gem

You can get it from json.rubyforge.org or just install it using:

  sudo gem install json

Install

To install with Git:

  mkdir -p ~/Library/Application\ Support/TextMate/Bundles
  cd ~/Library/Application\ Support/TextMate/Bundles
  git clone git://github.com/libin/Mojo.tmbundle.git "Mojo.tmbundle"
  osascript -e 'tell app "TextMate" to reload bundles'
